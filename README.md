# [ 카카오톡 챗봇 ]

### 주요 기능
 - 📢 /공지
 - ✅ /ㅊㅊ or /출첵
 - 🌦 /날씨 [지역]: 지역 날씨 검색
 - 👥 /정보 [이름]: 사용자의 채팅 정보 조회
 - 👤 /내정보     : 나의 채팅 정보 조회
   [2단계 명령어]
  - 🤖 /검색 [내용]: GPT를 활용한 내용 검색
  - 🏆 /순위    : 채팅 순위 조회
     [3단계 명령어]
    - 📰 /뉴스 [테마]: 최신 네이버뉴스 조회
	    [테마: 정치/경제/사회/생활/과학/세계]
    - 🎮︎ /게임    : 랜덤게임 실행
    - 🤷🏻 /힌트    : 랜덤게임 힌트
    - ⚐ /포기    : 랜덤게임 포기

미니게임 종류
 * 넌센스
 * 초성게임
 * 문장맞추기
 * 업다운
 * 야구게임
 * SQLD
 * 빅데이터분석기사 필기
